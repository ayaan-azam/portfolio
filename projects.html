Microsoft Access Database

For my BTEC I&CT, I created a doctors surgery database in MS Access

Firstly, I had to research how databases work and what industries they are used in.

I learnt key terms such as primary and foreign keys, as well as simple and comples queries.

I then created a doctors, patients, and appointment tables, linking them using 1 to many relationships.

I inputed sample data into these tables, and tested the database by making queries using SQL, and took action to improve the datbase with my test plan.

Python Database

As part of my BTEC I&CT course, I had to create a database in python, similar to my MS-Access one.

To do this, I used Pythons file-handling capabilities to create, add, remove and edit text files to store doctor, patient, and appointment information.

I also created features to search thorugh text files for specific items.

But firstly, I had to created a validation system, this was done using if and while loops. The same was done to create a main menu function

Additionally, I used string concatenation in order to make sure that all info entered into the text files were entered in order, like fields and records on a traditional database.
